float Controller::calculate_omega ( float torque_cmd_)

This function integrates torque to get speed.

This function uses a forward Euler integrator to integrate a commanded torque input to convert it to a speed.

Parameters
torque_cmd_The torque to integrate into a speed. The time used is the time between the last command and this command.
Returns
The calculated speed command in RPM, which is then passed to the state machine to command the motor.
